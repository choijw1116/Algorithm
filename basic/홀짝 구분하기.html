자연수 n이 입력으로 주어졌을 때 만약 n이 짝수이면 "n is even"을, 홀수이면 "n is odd"를 출력하는 코드를 작성해 보세요.

제한사항

1 ≤ n ≤ 1,000

  • ▣ 출력예시
  • ▣ 입력예제 1
    • 100
  • ▣ 출력예제 1
    • 100 is even
  • ▣ 입력예제 2
    • 1
  • ▣ 출력예제 2
    • 1 is odd
import java.util.Scanner; public class Solution { public static void main(String[] args) { Scanner sc = new Scanner(System.in); int n = sc.nextInt(); /* even = %2 == 0 odd = %2 != 0 */ if(n%2 == 0){ System.out.println(n + " is even"); }else if(n%2 != 0){ System.out.println(n + " is odd"); } } }
